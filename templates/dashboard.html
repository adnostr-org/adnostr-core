📚 AdNostr API Specifications

Core Endpoints

POST /api/v1/post_ad Generate Ad

Generate AI-powered advertisement and post to Mastodon

Payload: prompt, image_type, description, style, brand_elements
POST /api/v1/click_track Track Click

Track advertisement clicks for revenue attribution

Payload: post_id, click_source, campaign_id
GET /api/v1/ad/stream Get Stream

Get advertisement stream with expert data

Returns: Array of expert profiles with avatars/banners

Integration Examples

JavaScript Integration

// Generate advertisement
const response = await fetch('/api/v1/post_ad', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    prompt: "Beautiful sunset over mountains",
    image_type: "beauty",
    style: "photorealistic",
    brand_elements: ["nature", "adventure"]
  })
});

// Track clicks
await fetch('/api/v1/click_track', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    post_id: "123456",
    click_source: "tiktok",
    campaign_id: "summer_campaign"
  })
});
